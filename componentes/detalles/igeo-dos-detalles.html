Aplicacion web independiente desde cero. Spring Framework (MVC, Session, JPA-Data, Security) + PostgreSQL + Angular

Durante mi estancia en IGEO ERP Cloud Platform fui el encargado del desarrollo desde cero de un portal independiente que se interconectaba con la aplicación principal realizado desde cero

De esta forma se creó un proyecto Maven, trabajando con Spring Framework en la parte del servidor y con Angular en la parte del cliente.

En resumen se creo una aplicación web con manejo de usuarios mediante Spring Security, cuyas funcionalidades propias entre otras cosas podian interconectarse mediante un API a la aplicación principal, y cuya interfaz de cliente era totalmente manejado por Angular.
